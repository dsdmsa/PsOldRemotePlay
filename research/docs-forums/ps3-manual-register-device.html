Settings > Remote Play Settings > Register Device

Register Device

Register a device, such as a PSP™ system, to be used for remote play with the PS3™ system. Follow the on-screen instructions to register (pair) the device.

1.

On the PS3™ system, select (Settings) > (Remote Play Settings).

2.

Select [Register Device].

3.

Select the type of device to register.
Follow the on-screen instructions to complete the registration.

Hints

  • Types of devices that can be used for remote play and the availability of such devices vary depending on the country or region. For details on setting up a device for remote play, refer to the instructions supplied with the device.
  • You can use the USB cable (A - Mini-B type) supplied with the PS3™ system to connect the PSP™ system to the PS3™ system.
  • If you try to register a device for remote play that is registered with another PS3™ system, the registration (pairing) with the first PS3™ system will be cleared.
