# XDA Forums - PS3 Open Remote Play with Android
Source: https://xdaforums.com/t/ps3-open-remote-play-with-android.842398/

## Key Technical Findings

**Limitations Identified:**
- The Sony x10 was the only Android handheld with this capability at the time (2010)
- Remote Play described as "streaming music, pictures and certain types of videos to the registered device"
- Does not allow playback of disc-based games or movies without modification

**Implementation Requirements:**
1. First pairing with a PSP device
2. Extracting pairing information from a modified PSP
3. Using modded hardware to enable game streaming

**Protocol Considerations:**
- Sony's proprietary authentication system is the main barrier
- PS3 generates a specific password for connection
- Standard Android devices couldn't properly authenticate with this protocol

## Conclusion
No working solution for native Android Remote Play was achieved in this thread. The technical constraints were primarily Sony's closed protocol and lack of official Android app support.
