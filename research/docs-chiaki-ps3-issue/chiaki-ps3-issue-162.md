# Chiaki Issue #162 — PS3 Remote Play Support Request
Source: https://github.com/thestr4ng3r/chiaki/issues/162

## Maintainer Response (thestr4ng3r)

**Why PS3 is not supported:**
- "Remote Play on the PS3 is extremely limited compared to PS4. It is only supported in a handful of games, most of which are either tiny or available on other streamable platforms in better versions."
- "Also the protocol is an entirely different one." — The PS3 PREMO protocol is completely different from the PS4/PS5 protocol that Chiaki implements.

**Reference Implementation:**
- Pointed to open-source PS3 Remote Play implementation at `https://bitbucket.org/dsokoloski/open-rp` (original Open-RP project)

**Status:**
- No plans for integration
- Issue left open for community contribution
- Repository archived March 19, 2021 (read-only)

## Community Note
- megapro17 mentioned that PS3 Remote Play can be enabled on most games via custom firmware modifications (webMAN-MOD)
